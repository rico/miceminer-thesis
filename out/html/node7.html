Storing and Accessing the Data

The sheer amount of data expected to be collected by the automated system, has to be stored in a secure and stable way. This can be achieved by the use of a RDBMS (RDBMS) (see section on page ).

To access the data, an easy to handle user interface must be available, which facilitates the data exploration and export (see section on page ).
